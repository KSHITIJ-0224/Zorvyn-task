# 💹 Finance Dashboard - Modern Finance Management System

A sleek, high-performance finance management dashboard built with **React 19**, **Vite**, and **Material UI**. This application provides users with a comprehensive overview of their financial health, transaction tracking, and data-driven insights through interactive visualizations.

![Finance Dashboard Preview](https://via.placeholder.com/1200x600/0f0f0f/2dd4bf?text=Finance+Dashboard+Preview)

## ✨ Features

- **📊 Interactive Analytics**: Real-time data visualization using Recharts for tracking income, expenses, and savings trends.
- **💸 Transaction Management**: Comprehensive transaction logging with the ability to add, edit, and categorize financial activities.
- **🌑 Premium Dark Aesthetic**: A modern, high-contrast UI featuring glassmorphism and neon accents for a premium feel.
- **📱 Responsive Design**: Fully optimized for various screen sizes with a collapsible sidebar and fluid layout.
- **⚡ Performance First**: Built with Vite for ultra-fast Hot Module Replacement (HMR) and optimized build sizes.
- **🛡️ Type-Safe Architecture**: Fully implemented with TypeScript for robust code quality and developer experience.

## 🚀 Tech Stack

- **Core**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Styling**: [Material UI (MUI) 7](https://mui.com/) & [Emotion](https://emotion.sh/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd task-2
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

```text
src/
├── components/        # UI Components (Sidebar, TransactionModal, etc.)
├── context/           # State management using React Context API
├── data/              # Mock data and constants
├── assets/            # Static assets and images
├── types.ts           # Global TypeScript definitions
├── App.tsx            # Main application entry point
└── main.tsx           # React rendering root
```

## 🎨 Design System

The application uses a custom MUI theme defined in `App.tsx` featuring:
- **Primary Color**: `#2dd4bf` (Neon Teal)
- **Background**: `#070707` (Deep Black)
- **Surface**: `#0f0f0f` (Jet Black)
- **Typography**: 'Inter', sans-serif
